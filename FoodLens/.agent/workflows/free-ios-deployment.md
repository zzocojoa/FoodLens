# 📱 개인 기기 무료 설치 가이드

애플 유료 개발자 계정 없이도, 본인의 Apple ID만 있다면 실제 아이폰에서 앱을 실행해 볼 수 있습니다.

## 🛠️ 설치 준비

1. **Mac과 iPhone 연결**: 라이트닝 또는 USB-C 케이블을 사용하여 연결합니다.
2. **Xcode 실행**: 프로젝트 루트에서 `ios/FoodLens.xcworkspace` 파일을 엽니다.

## ⚙️ Xcode 설정 (Signing)

1. Xcode의 왼쪽 사이드바에서 **FoodLens 프로젝트 아이콘**을 클릭합니다.
2. **TARGETS** 섹션에서 `FoodLens`를 선택합니다.
3. 상단 탭에서 **Signing & Capabilities**로 이동합니다.
4. **Team** 항목에서 `Add an Account...`를 눌러 본인의 Apple ID로 로그인합니다.
5. 로그인이 완료되면 본인의 이름이 포함된 **Personal Team**을 선택합니다.
6. **Bundle Identifier**가 중복될 경우, 뒤에 임의의 숫자를 붙여 유일하게 만듭니다. (예: `com.yourname.foodlens.dev`)

## 🚀 앱 실행

1. Xcode 상단에서 실행 기기를 시뮬레이터가 아닌 **연결된 iPhone**으로 선택합니다.
2. **Run (▶️ 버튼)**을 클릭하여 빌드 및 설치를 진행합니다.

## 📡 외부 단독 실행 설정 (Release 모드)

기본 설정(Debug)은 앱 실행 시 항상 맥과 같은 와이파이에 연결되어 있어야 합니다. 맥 없이 밖에서도 앱을 실행하려면 다음 설정을 추가하세요.

1. **빌드 설정 변경**: Xcode 상단 툴바의 **Scheme (앱 아이콘 옆)** -> **Edit Scheme...** 클릭.
2. **Run** 항목 선택: 왼쪽 사이드바의 **Run** 메뉴 클릭.
3. **Info 탭**: **Build Configuration**을 `Debug`에서 `Release`로 변경.
4. **빌드 시작**: 다시 ▶️ 버튼을 눌러 앱을 설치합니다.
5. **확인**: 케이블을 뽑고 맥 없이도 앱이 즉시 켜지는지 확인합니다. (데이터 분석을 위해 LTE/5G는 필요합니다.)

## 🔓 기기에서 신뢰 설정 (최초 1회)

아이폰에서 앱이 설치된 후 바로 실행되지 않을 수 있습니다.

1. 아이폰의 **설정 (Settings) > 일반 (General)**으로 이동합니다.
2. **VPN 및 기기 관리 (VPN & Device Management)**를 선택합니다.
3. **개발자 앱 (Developer App)** 항목 스타일의 본인 Apple ID를 클릭합니다.
4. **'Apple ID' 신뢰하기**를 누릅니다.

## ⚠️ 중요 제한 사항

- **유효 기간 (7일)**: 무료 계정으로 설치한 앱은 **7일 후 만료**되어 실행되지 않습니다. 만료 시 다시 Mac에 연결하여 Xcode로 재설치(빌드)해야 합니다.
- **기기 제한**: 동일한 Apple ID로 로그인된 소수의 개인 기기에서만 설치가 가능합니다.
- **인터넷 연결**: 앱 설치 시점에는 Mac과 iPhone 모두 인터넷에 연결되어 있어야 인증이 완료됩니다.

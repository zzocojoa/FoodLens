# Food Lens 사업 계획서 (BM & 수익화 전략)

## 📊 비용 구조 (Cost Structure)

기존 이미지 생성 모델(GPU) 대비, **텍스트/멀티모달 추론 모델(Vertex AI Gemini)**은 비용이 획기적으로 낮습니다.

### 1회 분석 원가 (API 기준)

- **Gemini 2.0 Flash**: 약 **$0.00025 (약 0.35원)**
- **구성**:
  - 안전성 진단 (OCR + 알레르기 판별): ~$0.00013
  - 갤러리 모드 (전체 재료 탐색): ~$0.00012
  - _SAM 2 트래픽은 로컬/서버 처리 시 API 비용 없음 (서버 운영비 별도)_

> **결론**: 사용자 1명이 100번을 분석해도 원가는 **35원** 수준입니다. 마진율이 매우 높습니다.

## 💰 수익화 모델 (Revenue Streams)

여행객을 타겟으로 한 "유틸리티 + 정보성" 앱이므로, 단순 유료화보다는 **Freemium(부분 유료화)** 전략이 가장 유효합니다.

### 1. Freemium (구독/Pass 모델)

누구나 쉽게 접근하게 하되, "여행 기간" 동안 집중적으로 결제하도록 유도합니다.

- **Free (무료 사용자)**
  - 기능: 메뉴판 번역, 단순 음식명 인식.
  - 제한: 일 3회 분석 제한, "단순 주의(Caution)" 경고만 표시 (상세 성분 미제공).
  - 목적: 앱 설치 및 필수 앱으로 인식시키기.

- **Weekly Travel Pass (주간권: $4.99 / 약 6,900원)** ⭐ _핵심 상품_
  - 타겟: 1주일 내외 해외 여행객.
  - 기능:
    - **무제한 분석**: 식당마다 걱정 없이 찍어보기.
    - **Magic Touch / Gallery**: 숨겨진 재료 정밀 탐색 기능 잠금 해제.
    - **소통 카드**: 현지어 "이거 빼주세요" 카드 무제한 생성.
    - **오프라인 모드**: 미리 다운받은 알레르기 데이터베이스 연동 (가능 시).

> **수익성**: 주간권 1장($5) 판매 시, 해당 유저가 1,000번 분석($0.35)해도 **90% 이상의 이익**이 남습니다.

### 2. 제휴 & 광고 (Contextual Ads)

음식을 인식한다는 것은 **사용자의 "현재 위치"와 "관심사"**를 정확히 안다는 뜻입니다.

- **현지 맛집/쿠킹클래스 추천 (Affiliate)**
  - 상황: "Pad Thai(팟타이)" 인식 성공.
  - 액션: "이 음식이 마음에 드시나요? 근처 **쿠킹 클래스** 예약하기 (TripAdvisor 연동)"
  - 수익: 예약 건당 수수료 5~10%.
- **보험 연계**
  - 상황: "갑각류 알레르기 위험" 경고 발생.
  - 액션: "혹시 모를 응급 상황을 대비해, **여행자 보험**을 확인해보세요 (즉시 가입 링크)."

### 3. B2B 솔루션 (API Licensing)

- **타겟**: OTA(야놀자, Agoda, Triple 등) 앱 내 "맛집 기능"으로 SDK 공급.
- **모델**: API 호출 건당 과금 (Volume Pricing).
- 앱을 직접 마케팅하는 비용을 줄이고, 대형 플랫폼의 트래픽을 활용할 수 있습니다.

## 🚀 마케팅 전략 (Go-to-Market)

1.  **키워드 선점**: "동남아 여행 음식", "고수 빼주세요 앱", "해외 여행 알레르기".
2.  **커뮤니티 바이럴**: '태사랑', '유랑' 등 여행 커뮤니티에 "알레르기 환자의 태국 생존기" 콘텐츠 배포.
3.  **인플루언서**: 여행 유튜버에게 "현지 시장 음식 분석 챌린지" 협찬.

## 📝 요약

기술 비용(API)이 매우 낮으므로, **"많은 사용자 유치 -> 데이터 확보 -> 제휴 수익"** 또는 **"단기 고수익(여행 패스)"** 모델 모두 가능성이 높습니다. 개발 난이도 대비 가성비가 훌륭한 프로젝트입니다.
